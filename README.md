# Go-Rewrite

## Setup Steps

- Clone repo to local system.
- Copy and update `.env` file.
- In the Command Line:
  - `go get github.com/gorilla/mux`
  - `go get github.com/joho/godotenv`
  - `go get github.com/lib/pq`
  - `go test -v` to run tests.
